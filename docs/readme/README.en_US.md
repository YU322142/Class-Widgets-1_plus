## Features

- **Can run (via translation/emulation) on Loongson LoongArch processors, with compatibility close to the upstream project**
- **Smaller floating window**, taking less screen space
- A Python-based **plugin** system and plugin plaza (see latest builds)
- Displays today’s schedule in a **widget-based** style
- Supports [class start/end reminders](https://www.yuque.com/rinlit/class-widgets_help/fv2ou1i1ngap0hrl) and preparation bell, with TTS notifications
- Theme system with high customization
- Simple and intuitive [schedule editor](https://www.yuque.com/rinlit/class-widgets_help/oozelh8r56tmw0xb)
- Supports multiple schedule files, with import/export across Class Widgets
- Supports [**Course Schedule Exchange Standard** (CSES)](https://github.com/SmartTeachCN/CSES) for format conversion
- Provides quick [solutions](https://www.yuque.com/rinlit/class-widgets_help/gc4epffu7g5bf9os) for temporary schedule changes and class swaps
- Practical widgets such as “Weather” and “Custom Countdown”
- Personalize your Class Widgets through [“Custom”](https://www.yuque.com/rinlit/class-widgets_help/qyly70ht1ogge1pi)
- Supports Light/Dark themes and automatic switching based on system settings;<br>...

## Installation & Usage

> [!TIP]
> You may refer to the upstream [Class Widgets official documentation](https://www.yuque.com/rinlit/class-widgets_help/gs3gsbms1iivgibm).

Download the archive from Releases, extract it, and run `ClassWidgets.exe`.  
You can open settings or exit from the tray menu.
